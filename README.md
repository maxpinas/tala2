# Tala

Een React Native communicatie-app voor mensen met afasie.

## Beschrijving

Tala is een mobiele applicatie gebouwd met React Native en Expo, ontworpen om mensen met afasie te helpen communiceren door middel van:

- Zinsbouw met vooraf gedefinieerde woorden
- Snelle antwoorden
- Fotogalerij met bijschriften
- Medisch paspoort
- Noodcontact functionaliteit
- Partner uitleg scherm
